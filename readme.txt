Softwares needed : Maven, Spring Boot, Eclipse/Spring Tool Suite

Running the code through command line:
mvn spring-boot:run
more info. - https://spring.io/guides/gs/maven/#initial

Running the code through IDE
Run the Spring Boot application from IDE as a simple Java application, however, first import the project. Import steps will vary depending on IDE and build system. Most IDEs can import Maven projects directly, for example Eclipse users can select Import…? ? Existing Maven Projects from the File menu.

Memory Usage : HashMap to store the elements of the dictionary. Arraylist to store the final result

Time Complexty : Reading and storing the dictionary items need O(d) {where d is the number of terms in the dictionary}. Getting the scrabled words need again O(d) {Each and every word in the dictionary is compared to the requested word}.
